connector_groups

The following methods allow for interaction with the ZPA Connector Groups API endpoints.

Methods are accessible via zpa.connector_groups

class ConnectorGroupsAPI
get_group(group_id)

Get information for a specified connector group.

Warning

Deprecated since version 0.13.0: Use zpa.connectors.get_connector_group() instead.

Parameters:

group_id (str) – The unique identifier for the connector group.

Returns:

The connector group resource record.

Return type:

Box

Examples

>>> connector_group = zpa.connector_groups.get_group('2342342354545455')
list_groups(**kwargs)

Returns a list of all connector groups.

Warning

Deprecated since version 0.13.0: Use zpa.connectors.list_connector_groups() instead.

Returns:

List of all configured connector groups.

Return type:

BoxList

Examples

>>> connector_groups = zpa.connector_groups.list_groups()
